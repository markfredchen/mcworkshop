外汇TMBOX

    根据Trading EQ团队成员多年的交易经验，要实现外汇、商品期货等衍生品市场是长期稳定获利必须满足以下条件
    1、必须使用一套稳定可靠的程序化交易系统，因为机器交易可以规避人性弱点，严格执行交易纪律。另外由于外汇市场是24小时交易，人无法像机器一样24小时持续跟踪走势，不放过任何一个可能获得的机会。
    2、采用一套实时动态更新交易指标的自动化交易系统才可能长期在市场稳定获利。静态指标性交易的自动化交易是无法长期

稳定获得的，因为市场不断在变化，静态的指标很快会由于市场变化而失效。
    3、多策略多品种组合交易系统才有可能长期稳定获利，单一策略再优秀也会由于市场变化导致大比例的亏损。只有多策略多品种的自动交易系统才能通过识别市场形式匹配对应

策略才能稳定获利。
    4、资金管理才是稳定获利的关键，我们追求的是稳定的长期的获利模式，而不是短期的翻倍盈利。只有严格的资金管理体系，才能实现净值曲线稳定地增长，并在市场发生重大单边行情的是时候实现超额的收益。
