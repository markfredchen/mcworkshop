期货 CTA

    所谓CTA，是指商品交易顾问(Commodity Trading Advisor)，是期货行业中的一种基金管理方式。CTA的发展不仅为个人投资者减少期货投资风险提供了途径，而且也成为银行、对冲基金、共同基金及养老保险等机构投资者分散风险、扩展投资组合有效边界的工具。

    Trading EQ交易工房与Quant Technologies Limited宽客科技联合联合运作量化CTA团队，开发程序化交易产品，投资标的是国内与全球期货市场，涵盖外汇、农产品、贵金属、有色金属、能源、股指、利率与国债等，采用多策略组合模型，模型包括均值回归、趋势跟踪、波段交易、价差交易、对冲套利及日内高频系统等。

    量化CTA模式不是散户和机构投资者通过衍生品投资成功的唯一道路，但它一定是一条捷径！市场虽然是高度随机的，但并非完全随机，因而其中有一定的非随机因素存在。进行程式化交易就是为了方便交易者准确地、高效地捕获这些非随机因素，建立优势、提高交易获利的稳定性与一致性。

Trading EQ交易工房量化CTA产品的优势：

  • 减少交易中的主观随意性，及情绪的困扰

  • 利用策略概率及盈亏比的优势，创造可复制的收益

  • 减少人工下单错误机率、准确执行订单、降低交易损耗

  • 敏锐捕捉金融市场机会，5/24小时跟踪海内外市场波动

  • 及时回测新的交易思路与策略，降低投资组合策略之间的相关性，将投资组合的回撤降到最低

    Trading EQ交易工房的理念是投资就是和时间赛跑，通过有效多品种多策略降低投资组合的回撤与波动性，实现资产净值的稳定复利增长，因为复利增长才会铸就投资奇迹！
