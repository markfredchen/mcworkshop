客户问答

Q：Trading EQ（交易工房）具体提供投资者怎样的金融服务？


A：Trading EQ（交易工房）与Quant Technologies Limited宽客科技联合运作量化CTA团队，开发程序化交易产品。投资标的为国内以及全球期货市场，涵盖外汇、农产品、贵金属、有色金属、能源、股指、利率与国债等。采用多策略组合模型，模型包括均值回归、趋势跟踪、波段交易、价差交易、对冲套利及日内高频系统等。



Q：量化CTA产品的优势？


A：使用程序化交易在交易过程中可以克服人性的弱点，减少交易中的主观随意性，及情绪的困扰，严格执行交易纪律。利用策略概率及盈亏比的优势，创造可复制的收益；减少人工下单错误机率、准确执行订单、降低交易损耗敏锐捕捉金融市场机会。电脑是不折不扣的执行者，应当说几乎百分之百的做到知行合一。由于外汇市场是24小时交易，人无法像机器一样24小时持续跟踪走势，不放过任何一个可能获得的机会。Trading EQ（交易工房）的程序化交易产品5/24小时跟踪海内外市场波动及时回测新的交易思路与策略，降低投资组合策略之间的相关性，将投资组合的回撤降到最低。

Q：只有量化CTA模式才能在衍生品投资上取得成功吗？


A：量化CTA模式不是散户和机构投资者通过衍生品投资成功的唯一道路，但它一定是一条捷径！市场虽然是高度随机的，但并非完全随机，因为其中有一定的非随机因素存在。进行程式化交易就是为了方便交易者准确地、高效地捕获这些非随机因素，建立优势、提高交易获利的稳定性与一致性。



Q：Trading EQ（交易工房）的智能交易TMBOX相较于市场上的 大众化的外汇EA，有什么区别？


A：TMBOX真正做到了实时动态更新交易指标，多策略多品种交易匹配，严格控制风险的资金管理系统。采用一套实时动态更新交易指标的自动化交易系统才可能长期在市场稳定获利。静态指标性交易的自动化交易是无法长期稳定获得的，因为市场不断在变化，静态的指标很快会由于市场变化而失效。多策略多品种组合交易系统才有可能长期稳定获利，单一策略再优秀也会由于市场变化导致大比例的亏损。只有多策略多品种的自动交易系统才能通过识别市场形式匹配对应策略才能稳定获利。
